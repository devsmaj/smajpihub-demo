# SMAJ STORE App\n\nA standalone copy of the SMAJ STORE experience that now lives under pps/smaj-store. It ships with the same HTML, CSS, and JavaScript from the original project, but the logo is centralized inside ssets/images/logo.png.\n\n## Structure\n- index.html is the homepage with hero, categories, featured/trending/recent products, top vendors, banner, and footer content.\n- pages/ carries all of the secondary store pages (cart, checkout, orders, vendor views, etc.).\n- css/styles.css, js/app.js, js/cart.js, and js/products.js power the layout, PI HUB wallet integration, cart behavior, and mocked product data.\n- ssets/images/logo.png holds the store badge referenced by every page copy.\n\n## Preview\nOpen pps/smaj-store/index.html (or run any static server from the pps/smaj-store directory) to see the store in action.
